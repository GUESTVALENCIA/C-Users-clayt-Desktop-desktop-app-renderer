4. ENDPOINTS DEL SERVIDOR MCP (Puerto 19875):
   - GET http://localhost:19875/tools - Lista todas las herramientas disponibles
   - POST http://localhost:19875/call - Ejecuta una herramienta
     Body: { "tool": "nombre_herramienta", "params": {...} }
   - GET http://localhost:19875/memory - Lista la memoria local

5. HERRAMIENTAS DISPONIBLES (Puerto 19875):
   - read_file: Leer archivos del sistema
   - write_file: Escribir archivos en el sistema
   - list_files: Listar archivos y directorios
   - execute_command: Ejecutar comandos CMD/PowerShell
   - memory_store: Guardar en memoria local
   - memory_get: Obtener de memoria local
   - memory_list: Listar toda la memoria
   - memory_search: Buscar en memoria

6. HERRAMIENTAS DEL SERVIDOR NEON (Puerto 8765):
   - get_memory: Obtener memoria persistente de NEON
   - set_memory: Guardar memoria persistente en NEON
   - search_memory: Buscar en memoria NEON
   - get_chat_history: Obtener historial de conversaciones

7. ARCHIVOS IMPORTANTES:
   - main.js: Proceso principal de Electron
   - studiolab-final-v2.html: Interfaz principal de la aplicación
   - mcp-server.js: Servidor MCP de herramientas (puerto 19875)
   - mcp-server-neon.py: Servidor MCP de memoria persistente (puerto 8765)
   - qwen-mcp-stdio-server.js: Servidor MCP stdio para app oficial de QWEN

8. NOTAS:
   - Los servidores MCP se inician automáticamente cuando se ejecuta la app Electron
   - Ambos servidores están en localhost (127.0.0.1)
   - No hay CORS restringido (Access-Control-Allow-Origin: *)
   - Los servidores están disponibles mientras la aplicación Electron esté ejecutándose
